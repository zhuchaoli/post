# 用Java实现的设计模式

设计模式是程序员在设计应用程序或系统时可以用来解决常见问题的最佳形式化实践。

设计模式可以通过提供经过测试的、经过验证的开发范例来加速开发过程。

重用设计模式有助于防止导致重大问题的细微问题，并且还提高了熟悉这些模式的编码人员和架构师的代码可读性。

**传送门**：https://github.com/iluwatar/java-design-patterns
